# Aptos Wallet Adapter - Vue/Nuxt

Vue and Nuxt.js integration for Aptos Wallet Adapter.

## Packages

- **[@aptos-labs/wallet-adapter-vue](./packages/wallet-adapter-vue)** - Vue 3 integration for Aptos Wallet Adapter

## Example Applications

- **[nuxt-example](./apps/nuxt-example)** - Nuxt 3 demo application

## Getting Started

### Installation

```bash
pnpm add @aptos-labs/wallet-adapter-vue @aptos-labs/wallet-adapter-core
```

### Usage

See the [wallet-adapter-vue README](./packages/wallet-adapter-vue/README.md) for detailed usage instructions.

## Development

This is a pnpm monorepo using Turbo for build orchestration.

### Prerequisites

- Node.js >= 20.0.0
- pnpm 9.15.5

### Setup

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm build

# Run development server
pnpm dev
```
